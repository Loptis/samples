# Spring Boot Shiro Sample

**Sample project for spring boot shiro.**

**本项目是一个spring boot集成shiro权限框架的示例项目**
## Features

- 使用mysql数据库
- 集成mybatis plus
- 使用redis作为缓存机制，存储用户session及权限

## Getting started

- 安装mysql数据库，执行sample_shiro.sql脚本
- 配置mysql连接信息
- 本地安装redis
- Run SampleSpringBootShiroApplication
